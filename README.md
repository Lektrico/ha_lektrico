# Lektrico Charging Station
This custom integration allows you to manage your [Lektrico EV Charger](https://lektri.co/charging-solutions/1p7k/).

## Installation ##
Configuration can be done on UI, you need to enter the IP of your charger.

## Supported entities ##
- Charger State
- Charging Time
- Current
- Dynamic Current
- Install Current
- Install Power
- Led Brightness
- No Authentication
- Session Energy
- Temperature
- Total Charged Energy
- Voltage

## Device Info ##
- Name
- Serial Number
- Revision
- Producer
- Firmware
